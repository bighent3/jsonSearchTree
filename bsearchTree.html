This is a simple HTML + JavaScript web application that allows a user to input personal information (name, age, address, phone) and stores each user in a Binary Search Tree (BST) based on their name. The application provides two main functionalities via buttons: Add User: Prompts the user for input and adds them to the BST. Display Sorted Users: Displays the list of users sorted alphabetically by name using an in-order traversal of the BST. Interactive User BST
